# 🇮🇳 Indian Startup Funding Analysis & Prediction

## 📌 Project Overview

This project analyzes the Indian Startup Funding dataset using Python and Machine Learning. It performs data cleaning, exploratory data analysis (EDA), data visualization, and builds a regression model to predict startup funding amounts.

The project is implemented in Google Colab/Jupyter Notebook using popular data science libraries.

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Funding Trends Over Time
- Top Startup Sectors
- Top Funded Startups
- Top Cities Receiving Investments
- Active Investors Analysis
- Investment Type Distribution
- Funding Amount Distribution
- Sector vs City Heatmap
- Summary Statistics
- Machine Learning Regression Model
- Feature Importance Visualization
- Model Saving using Joblib

## 📊 Machine Learning Model

The project uses a Regression Model to estimate startup funding amount based on features such as:

- Industry Vertical
- City Location
- Investment Type
- Year

The preprocessing pipeline includes:

- Missing value handling
- One-Hot Encoding
- Feature Transformation

The trained model is saved as a `.joblib` file for future predictions.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Indian-Startup-Funding-Analysis.git
```

2. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn scikit-learn joblib
```

3. Open the notebook

```bash
jupyter notebook IndianStartup_fixed.ipynb
```

or upload it to **Google Colab**.

4. Place the dataset in the correct location and run all notebook cells.
